Sequential Programs


  • C is a Sequential language or procedural language because the execution flow is always in a sequence.
  • In general, given the same input data, a sequential program will always execute the same sequence of instructions and it will always produce the same results.

    10.Write a program to convert temperature Fahrenheit to Celsius.



  • C program:

    • #include<stdio.h>
    #include<conio.h>

    void main()
    {
    float F,C;
    clrscr();
    printf("\nEnter Value in Fahrenheit:");
    scanf("\n%f",&F);

    C=5.0/9*(F-32);

    printf("\nCelsius Value:%.2f",C);
    getch();
    }


  • Output:

    • Enter Value in Fahrenheit:40

    Celsius Value:4.44

    11.Write a program that input quantity, price, discount and display final amount.



  • C program:

    • #include<stdio.h>
    #include<conio.h>

    void main()
    {
    float Q,P,D,disc,cost,TP;
    clrscr();
    printf("\nEnter Quantity:");
    scanf("\n%f",&Q);

    printf("\nEnter Price:");
    scanf("\n%f",&P);

    printf("\nEnter Discount:");
    scanf("\n%f",&D);

    TP=Q*P;
    disc=(TP*D)/100.0;
    cost=TP-disc;

    printf("\nFinel Amount:%f",cost);
    getch();
    }


  • Output:

    • Enter Quantity:2

    Enter Price:100

    Enter Discount:50

    Finel Amount:100

    13.Write a program to input height and breadth and find out area of triangle.



  • C program:

    {
